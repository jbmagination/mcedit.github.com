class mceditlib.anvil.adapter.AnvilSection(section_tag=None)[source]¶

Bases: object

Internal representation of a 16x16x16 chunk section. Arrays are indexed YZX.

4-bit arrays are unpacked to byte arrays to make them work with numpy’s array routines.

To create the full 12-bit block ID, the Blocks array is extended to 16 bits and the Add array is merged into the Blocks array.

Variables:
  • Y – section’s Y value [0..(world.Height+15) >> 4]
  • Blocks – Block IDs [0..4095]
  • Data – Block sub-data [0..15]
  • BlockLight – Light emitted by blocks [0..15]
  • SkyLight – Light emitted by the sun/moon [0..15]
